Welcome to the Phonebook API Documentation

This page provides information on the available API endpoints for the Phonebook application.

Available Endpoints

  • POST /api/register/ - Register a new user
  • POST /api/login/ - Login a user and obtain a JWT token
  • POST /api/mark_as_spam/ - Mark a phone number as spam
  • GET /api/search-by-name/ - Search for a contact by name
  • GET /api/search-by-phone/ - Search for a contact by phone number
  • POST /api/add-contact/ - Add a new contact
  • GET /api/get-contacts/ - Get all contacts for the logged-in user
  • POST /api/token/refresh/ - Refresh the JWT token

Use these API endpoints to interact with the Phonebook application. Ensure you are authenticated (logged in) before accessing most of these resources.
